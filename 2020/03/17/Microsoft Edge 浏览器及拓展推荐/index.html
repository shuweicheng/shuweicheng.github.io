Microsoft Edge 浏览器及拓展推荐

账号轻松同步，插件丰富，启动快速，内存占用少

2020最值得推荐的浏览器毫无疑问是新版的Microsoft Edge 浏览器。在此之前，相信绝大部分人心中最好用的浏览器是谷歌Chrome浏览器，它简洁、快速、稳定、而且有着丰富的拓展插件。但是也很遗憾，Chrome在国内使用有个明显的弊端：不借助翻墙插件无法同步谷歌账户和进入谷歌应用商店，这也就有了“先有鸡还是先有蛋”的问题。基于谷歌 Chromium 内核的全新浏览器 Microsoft Edge 浏览器，==具有像Chrome 浏览器一样的高效稳定的特点，而且能够轻松同步微软账户和打开微软应用商店下载各种插件，并且具有更低的内存占用空间==。

  • 在更换一个浏览器时，大家一定会考虑迁移的问题，Microsoft Edge 浏览器支持从其他浏览器导入数据，因此替代成本低。
  • 下载安装新的chromium内核Edge浏览器时，会自动替代win10自带的旧版本Edge 浏览器
  • 目前Edge也可以直接安装谷歌应用商店的插件，但是反过来不行
  • 小缺点：目前还无法同步插件、历史记录
  • 下载地址：Microsoft Edge 浏览器

必装拓展插件18个推荐。（微软应用商店有时打开比较慢，耐心等待）

  1. 谷歌上网助手

首装神器

科学上网，可以访问谷歌google搜索，gmail邮箱，google+等谷歌产品以及维基百科。

  • 下载地址：谷歌上网助手 - Microsoft Edge Addons
  1. Infinity 新标签页（Pro）

界面美观，精美壁纸，多搜索引擎切换，备份同步功能

自定义标签页，可以将网页装扮成类似安卓的应用，支持多种搜索引擎切换，支持切换各种精美壁纸，具有 数据备份同步功能。

  • 下载地址：Infinity 新标签页 (Pro) - Microsoft Edge Addons

  1. 沙拉查词-沙拉查词-聚合词典划词翻译

超级好用的划词翻译，集合了谷歌翻译和有道翻译

Edge 自带 bing 全文翻译，配合沙拉划词翻译，有效缓解英文阅读压力。而且拓展还支持单词本功能和查询 历史记录功能。

  • 下载地址：沙拉查词-聚合词典划词翻译 - Microsoft Edge Addons

  1. Adblock Plus - 免费的广告拦截器

老牌广告拦截器

阻止 广告、弹出窗口并抵御恶意软件，支持自定义软件，“世界终于清净了”。

  • 下载地址：AdBlock — 最佳广告拦截工具 - Microsoft Edge Addons
  1. LastPass: Free Password Manager

免费的密码管理器，跨平台同步，自动登录

只需记住一个账号密码，便可以管理所有账号密码，解决了设置密码简单，记账号密码难的痛点。

  • LastPass只有密码提示，无法找回密码，一定要记住主密码！！！

  • 安卓版本已退出中国市场

  • 下载地址：LastPass: Free Password Manager - Microsoft Edge Addons

  1. 图片助手(ImageAssistant) 批量图片下载器

效率神器，找图必备

一键提取网页内图片，不需登录直接下载，找图必备，谁用谁知道。

  • 下载地址：图片助手(ImageAssistant) 批量图片下载器 - Microsoft Edge Addons
  1. 网页截图:注释&录屏

优秀的截图插件

捕获任何网页的全部或部分。添加注释，评论，模糊敏感信息，以及一键上传。还有免费的桌面捕获功能！

  • 下载地址：网页截图:注释&录屏 - Microsoft Edge Addons
  1. 京价保

京东薅羊毛，懒癌必备

主要功能：自动监控最近订单商品的价格变化，在商品降价时自动申请价格保护，自动签到领取京豆，自动签到领取钢镚，自动领Plus券，自动领全品类，商品价格走势图（仅支持京东）

  • 下载地址：京价保 - 京东价保助手 - Microsoft Edge Addons
  1. 喵喵折

全网比价插件

全网比价，货比三家不吃亏。

  • 下载地址：喵喵折 - Microsoft Edge Addons
  1. 印象笔记·剪藏/Evernote Web Clipper

搭建个人知识体系

使用印象笔记·剪藏扩展程序一键保存精彩网页内容到印象笔记帐户，不用复制粘贴编辑再整理，信息收集快人一步，搭建个人知识库。

  • 非会员只有两台电脑的授权

  • 下载地址：印象笔记·剪藏 - Microsoft Edge Addons

  1. OneNote Web Clipper

免费的笔记管理软件

和印象笔记·剪藏一样的功能，可以保持网页内容至OneNote软件中，OneNote为Win10系统自带软件，界面美观，层级多。

  • 虽然能正常同步，但是无法登陆OneDrive系统进行笔记本删除。

  • 下载地址：OneNote Web Clipper - Microsoft Edge Addons

  1. Google学术搜索按钮

学术必备插件

科研必备，快捷方便。配合Sci-hub，快速下载论文。

  • 下载地址：Google学术搜索按钮 - Microsoft Edge Addons
  1. 一键视频下载器

一键下载网页中的视频

是否看到网页、微博上的视频很想保存留念或使用，本拓展就可以突破网站限制，直接下载你要的小视频。

  • 下载地址：一键视频下载器 - Microsoft Edge Addons
  1. 哔哩哔哩(Bilibili)播放器扩展

实用的 B 站拓展

  • 自动播放

  • 关灯设置（就是视频之外界面灰掉）

  • 字幕设置

  • 下载地址：哔哩哔哩(Bilibili)播放器扩展 - Microsoft Edge Addons

  1. Bilibili哔哩哔哩下载助手

B 站视频下载

B 站视频直接下载保存，bingo~

  • 下载地址：bilibili哔哩哔哩下载助手 - Microsoft Edge Addons
  1. VIP会员视频

没有VIP会员的将就办法

VIP会员视频通过解析网站，支持优酷、爱奇艺、搜狐、腾讯等多个视频平台。VIP会员电影免费看，告别充值会员的烦恼。但不能点播用点券的VIP电影。

  • 下载地址：VIP会员视频 - Microsoft Edge Addons
  1. Tampermonkey

最好用的脚本管理器，突破网站限制

Tampermonkey 也就是大家常说的“油猴”，其实是一个脚本管理器，类似于拓展插件，通过一段脚本代码来优化网页体验，突破一些网站的限制。脚本网站推荐 Geasy Fork 。

  • 下载地址：Tampermonkey - Microsoft Edge Addons
  1. Listen 1

全网音乐

Listen 1可以搜索和播放来自网易云音乐，虾米，QQ音乐，酷狗音乐，酷我音乐网站的歌曲，让你的曲库更全面。

  • 下载地址：Listen 1 音乐播放器
