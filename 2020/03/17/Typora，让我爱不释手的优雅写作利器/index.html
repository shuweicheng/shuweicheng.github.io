Typora，让我爱不释手的优雅写作利器

简介

  • Typora 是一款支持实时预览的 Markdown^1 文本编辑器，所见即所得。
  • 简洁优雅、设计美妙、功能强大，被誉为“世界上最美的写作APP“
  • OS X、Windows、Linux 多平台免费。

官方主页及下载地址：Typora — a markdown editor, markdown reader.

常用快捷键

功能 快捷键 Markdown语法
加粗 Ctrl + B ** **
斜体 Ctrl + I * *
下划线 Ctrl + U
表格 Ctrl + T
一级标题，二级标题，···六级标题 Ctrl + 1,2,···6，以此类推 #
启用源代码模式 (切换原文/Markdown) Ctrl + /
撤销 Ctrl + Z
代码块 Ctrl + shift + K ```
有序列表 Ctrl + shift + [ 1.
无序列表 Ctrl + shift + ] -
引用 Ctrl + shift + Q >
段落 Ctrl + 0
增加缩进 Ctrl + ]
减少缩进 Ctrl + [
清除样式 Ctrl + \

常用快捷键与 Word 相似，新手刚开始不必急于记忆 Markdown 语法。

建议平常写作，多用快捷键，效率提升明显。

特色功能

丰富主题

Typora 有着美观丰富的主题库，也可以自定义主题。主题也就是富文本编辑器的排版，通过更换主题可以实现各种漂亮的排版。这也就意味着同一个 Markdown 的文档，可以被渲染成各种排版格式，然后可以直接粘贴到微信公众号、简书、知乎、今日头条等平台，或印象笔记、Onenote等笔记软件当中。

插入表情符号

在 Typora 中，你可以用 :emoji: 的形式来打出 emoji，软件会自动给出图形的提示。例如，

:dog:(​:dog:​)

:girl:(:girl:)

插入视频

源码格式： <video src="https://www.typora.io/img/beta.mp4"></video>

多种格式导出

Typora 原生支持导出 PDF，HTML等格式。你可以根据软件内提示安装 Pandoc 插件来导出更多例如 docx，LaTeX 等格式。

常见问题

  • 图片和视频插入

Typora 插入图片和视频的方式有三种：

  1. Markdown 语法插入；
  2. 复制图片后直接粘贴；
  3. 拖动本地图片或视频到文档位置；

后面两种方式，Typora会自动插入符合Markdown 语法的语句。

图片和视频插入时有个问题需要注意，Markdown实际上插入图片实际上都是插入 Markdown 语法的图片链接，因此保存了本地图片的 md 文件在发送给其他人时，别人是无法正常显示图片的。所以，建议大家通过使用图床[^2]功能，来保证文档在分享后仍能正常显示。

脚注

[^2]: 图床一般是指储存图片的服务器，可以参考PicGo+GitHub：你的最佳免费图床选择！ - 知乎
